# 角斗士赌局 — 游戏规则

## 游戏概述

两名玩家进行为期三天的赌局，每天进行 3 场比赛。
每天，两名玩家各自组建 3 人角斗士小队，通过暗标拍卖获得角斗士，然后派出角斗士进行 1v1 对战。3 天结束后，奖励池 point 按 1:1 兑换为游戏币，最终按游戏币总额兑换现金。

每位玩家初始拥有 **800 游戏币**。拍卖出价和系统补齐均从游戏币扣除，不足时从奖励池补扣。

### 游戏币与奖励池

- **游戏币**：用于拍卖出价和系统补齐的即时货币，初始 800，可随时使用。
- **奖励池（死钱）**：存储比赛结算后获得的 point，**完全不能用于拍卖出价**。唯一用途：(1) 系统补齐时游戏币不够 → 从奖励池补扣；(2) 每日胜者可转最多 50 point → 游戏币；(3) 3 天结束后 1:1 兑回游戏币。**核心：奖励池在比赛期间是冻结资产，不能用来竞拍角斗士。**

## 疲劳机制

角斗士每次出战后会产生疲劳，影响下一场比赛的 HP 倍率：

| 疲劳等级 | HP 倍率 | 说明 |
|---------|---------|------|
| 1.0 (满血) | 100% | 未出战过，或充分休息 |
| 0.9 | 90% | 曾出战，昨天已休息恢复 |
| 0.8 | 80% | 昨天出战过 |
| 0.6 | 60% | 连续两天以上出战 |

休息一天可提升一级疲劳（0.6→0.8→0.9→1.0）；出战则降低一级（1.0→0.8→0.6）。

## 每日流程

### 0. 赛前展示（仅第 1 天）

展示全部 20 名角斗士的胜率排名（从高到低），供玩家策略参考。
**重要提醒：胜率排名仅反映历史胜率，拍卖池中角斗士顺序是随机排列的，暗标时也不显示胜率，需凭经验判断。**

> 胜率数据来源：每两名角斗士之间对战 200 场，每名角斗士总计战斗 3800 场得出的胜率。

### 1. 情报阶段

每位玩家获得若干名随机角斗士的**真实历史胜率数据**（不重复展示已看过的）。第 1 天 5 名，第 2 天 4 名，第 3 天 3 名。

情报格式为：**名称、char_id、胜场数、总场数、平局数、综合胜率（百分比）**。不是对特定对手的胜负关系，而是该角斗士对所有对手的历史总胜率。
**注意：两名玩家的情报各自独立随机抽取，互不影响。**

### 2. 拍卖阶段

- 从 20 名角斗士中随机抽取 **9 名**进入拍卖池，**逐个展示**。
- 展示时只显示角斗士的名称和当前疲劳值，**不显示胜率**。
- 双方同时进行**暗标出价**（私下决定，互不知晓对方出价），然后由主持者输入系统比价。
- 出价范围：**0（弃权）或 50~150 游戏币**。
- **双方均扣除自己的出价金额**（无论谁赢）。即：A 出 50、B 出 60 → A 扣 50，B 扣 60，B 获得角斗士。
- **输方的游戏币直接转入输方自己的奖励池**：以上例，A 出 50、B 出 60 → A 扣 50 并转为 A 自己的奖励池，B 正常消耗 60。
- **⚠️ 奖励池是死钱！** 奖励池 point 不能用于拍卖出价。输掉拍卖 = 你的游戏币从活钱变成了死钱——它只能在系统补齐时当后备、每日胜者转最多 50、或最终兑换时才有用。
- 出价相同则重拍（最多重拍 1 次，再次相同则跳过该角斗士）。**平局重拍不扣游戏币**，只有一方胜出时才会扣除双方出价。
- 双方均出 0 则跳过该角斗士。
- 当一方率先获得 **3 名**角斗士后，另一方**立即失去拍卖权**，系统自动以 **85 游戏币/个** 从拍卖池剩余未认领的角斗士中随机抽取补齐至 3 人。
- **角斗士仅在当天属于你，第二天全部回到拍卖池，重新竞拍**。

#### 资金不足与破产

- 当一方游戏币 **< 50** 时，该玩家在后续拍卖中只能弃权（出价强制为 0），对方可按起拍价 50 逐次决定是否竞拍当前角斗士。
- 系统自动补齐时，逐个扣除 85 游戏币/人，**优先扣游戏币，不够再从奖励池补扣**。每扣一人前检查 `游戏币 + 奖励池 ≥ 85`，不足则**立即破产**：游戏币清零并全部归对方，未补的空位不再补齐，**游戏立即结束**。系统**不会让游戏币变成负数**。

### 3. 比赛阶段

- 双方各从自己的 3 名角斗士中**选 1 名**出战。
- 角斗士按当前疲劳值的 HP 倍率进入战斗。
- 比赛由 Arena 引擎 1v1 无渲染对战，限时 120 秒，超时判平。
- 每名角斗士每天只能出战一次。

### 4. 结算阶段

#### point 机制

- 拍卖获得角斗士时，角斗士携带 `point = 成交价`（即该玩家的出价金额）。
- 比赛后，**胜方**从败方 point 中夺取 `min(胜方point, 败方point)` 数量的 point。
- 胜方自身的 point + 夺取的 point → 胜方奖励池；败方剩余的 point → 败方奖励池。
- 平局/超时：双方各自 point 原样归入各自奖励池。
- 结算后角斗士 point 清零。

#### 每天第二轮特殊效果

每天第 2 场比赛，胜方夺取量 **×1.5**（即 `min(胜方point, 败方point) × 1.5`）。
败方扣除夺取量后，奖励池**可能为负数**。

示例：
- A(100pt) 胜 B(60pt) → A 获 100+90=190，B 获 60-90=-30
- A(60pt) 胜 B(100pt) → A 获 60+90=150，B 获 100-90=10

#### 胜负效果速查

| 轮次 | point 转移 | 胜方奖励池 | 败方奖励池 |
|------|-----------|----------|----------|
| 第1、3轮 | 夺取 min(胜方, 败方) | +胜方point + 夺取 | +败方point - 夺取 |
| 第2轮 | 夺取 min(胜方, 败方) × 1.5 | +胜方point + 夺取 | +败方point - 夺取（可为负） |
| 平局/超时 | 无转移 | +己方point | +己方point |

#### 疲劳更新（每场比赛后）

- 出战角斗士疲劳恶化：1.0→0.8, 0.9→0.8, 0.8→0.6, 0.6→0.6。
- 未出战的角斗士当天疲劳不变，在每日结算时统一恢复。

#### 每日胜者奖励（3 场比赛全部结束后）

- 判定标准：当天**胜场数**多者为本日胜者。胜场数相同则无奖励。
- 胜者的奖励池 point 可按以下规则转换为游戏币：

| 奖励池 point | 转换规则 |
|-------------|---------|
| ≤ 0 | 无转换 |
| 1 ~ 49 | **全部**转为游戏币 |
| ≥ 50 | 转换 **50** point 为游戏币 |

#### 每日结算（每日胜者奖励之后）

- 所有未出战角斗士疲劳恢复一级：0.6→0.8, 0.8→0.9, 0.9→1.0, 1.0→1.0。
- 出战角斗士疲劳不再变化（已在每场比赛后更新过）。

---

## 最终胜负

3 天全部结束后：
- 双方奖励池 point 按 **1:1 兑换为游戏币**。
- 比较双方**游戏币总额**，高者获胜。
- 中途有玩家破产则对方直接获胜。
