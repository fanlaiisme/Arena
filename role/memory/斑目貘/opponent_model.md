## 出价模式
- 激进程度: medium-high — 第一轮对snowman出120（激进），但第二轮对monk出价低于70（保守）
- 选择性: 高 — 对特定角斗士（snowman）愿意出高价，对其他角斗士则保守或可能弃权
- 偏好出价区间: 愿意为看重的角斗士出110~120+的高价，对不看重的不超过69；对未知角斗士试探性出价70左右，缺乏情报时趋于保守
- 虚张声势倾向: 待观察
- 弃权率: 待观察（第2轮可能弃权或低出价）
